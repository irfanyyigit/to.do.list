# To-Do List Application

---

## Description

A modern and user-friendly to-do list application designed to help users efficiently manage their daily tasks and improve productivity. This web-based application provides an intuitive interface for organizing, tracking, and prioritizing tasks with ease.

The To-Do List Application combines functionality with elegant design, featuring soft coffee tones that create a comfortable and distraction-free environment. Whether you are managing personal tasks, work projects, or daily errands, this application helps you stay organized and focused on what matters most.

The primary objective of this project is to provide users with a simple yet powerful task management tool that promotes productivity, organization, and effective time management through a clean, accessible, and visually appealing interface.

---

## Technologies

The following technologies were utilized in the development of this application:

- **HTML5** - Semantic markup and responsive structure
- **CSS3** - Modern styling with smooth animations and transitions
- **JavaScript (ES6+)** - Dynamic task management and user interaction
- **Google Fonts** - Professional typography (Poppins and Lora)

## Features

- **Add Tasks** - Quickly add new tasks with a single click or by pressing Enter
- **Delete Tasks** - Remove completed or unwanted tasks instantly
- **Edit Tasks** - Modify existing tasks to update information
- **Mark Complete** - Toggle task completion status with visual feedback
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Soft Coffee Aesthetics** - Calming color palette reduces eye strain
- **Intuitive Interface** - Minimal and user-friendly design
- **Real-time Updates** - Immediate visual feedback for all actions

## Installation

Getting started with the To-Do List Application is simple:

1. **Clone or download** this repository to your computer
2. **Navigate** to the project folder
3. **Open** the landing page (`index.html`) in any modern web browser
4. **Click** "Go to Application" to access the main to-do list interface

**No installation or dependencies required** — this is a standalone web application that runs entirely in your browser with local storage support.

---

## Usage

Using the To-Do List Application is straightforward:

1. **Open** the application in your web browser
2. **Enter** your task in the input field at the top
3. **Add Task** by clicking the add button or pressing Enter
4. **Manage Tasks** using the action buttons:
   - **Checkmark (✔)** - Mark task as completed
   - **Edit (✎)** - Modify the task text
   - **Delete (🗑)** - Remove the task from your list
5. **View Progress** - See completed tasks displayed with visual distinction

## How It Works

The To-Do List Application uses a straightforward workflow:

1. **Input** - User enters a task description in the input field
2. **Creation** - New task element is dynamically created in the DOM
3. **Management** - User can complete, edit, or delete tasks
4. **Feedback** - Visual feedback for all interactions (toggling, deletion, updates)
5. **Interaction** - Supports both click events and keyboard shortcuts (Enter key)

All task management occurs on the client-side for instant responsiveness and privacy.
